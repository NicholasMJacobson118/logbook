Google Drive sync
100% free — uses Google Apps Script, which is built into every Google account. No billing, no API keys, no cloud setup fees.
One-time setup (~5 minutes)
1
Open Google Apps Script
Go to script.google.com and click New project. Sign in with the Google account where you want your logbook saved.
2
Paste the backend script
Delete all the default code in the editor. Open apps_script.gs (the other file you downloaded), copy everything, paste it in. Press Ctrl+S to save and give the project a name like Logbook Sync.
3
Deploy as a web app
Click Deploy → New deployment (top right).
Set type to Web app.
Set "Execute as" to Me.
Set "Who has access" to Only myself.
Click Deploy → authorize when prompted → copy the Web app URL.
4
Paste your Web app URL here
Sync your entries

Pushes all entries to a Google Sheet called Dual Role Logbook in your Drive. Running it again updates everything — no duplicates created.
